    description du projet
    Le présent mémoire s'inscrit dans le cadre du projet java EE S6 portant sur la gestion des taxes séjours, pour les communes. Pour mener à bien notre projet, nous avons entamé notre travail par une étude fonctionnelle détaillée en premier lieu, en second lieu, nous nous sommes focalisés sur une étude technique pour maitriser les langages et les outils de développement à utiliser. Une fois maitriser le périmètre de notre projet, nous avons choisi la démarche Scrum pour mieux cerner notre problématique de conception et de réalisation du module.
